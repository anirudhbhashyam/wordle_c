# WORDLE 

![build](https://github.com/anirudhbhashyam/wordle_c/actions/workflows/build.yml/badge.svg)

A Wordle implementation in `c` using the terminal. 

# Usage 
```
>>> ./pack.sh run
```
or 
```
>>> bash pack.sh run
```

# Docker Run
```
>>> docker build -t wordle .
>>> docker run --rm -it wordle
```

# Dependencies 
    - `gcc`
